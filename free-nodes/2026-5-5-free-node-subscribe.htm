什么是科学上网？

科学上网，顾名思义，指的就是通过特定的技术手段或软件工具，帮助用户绕过各种网络限制，以实现自由、安全的互联网访问。在某些国家或地区，由于政治、文化或其他原因，很多全球知名的网站和服务（如Google、YouTube、Facebook等）受到严格的访问限制，无法直接访问。此时，科学上网工具就成为了用户突破这些限制、享受全球互联网资源的必备工具。

科学上网工具的核心原理是通过加密技术、代理技术或者VPN技术，将用户的网络流量转发到其他可访问区域的服务器，从而实现用户访问被限制内容的目的。

FAQ

Netch 的 TCP/UDP 分流适合哪些应用？
TCP 分流适合网页、下载和视频播放，UDP 分流适合游戏、语音和实时视频。Netch 可同时代理 TCP 和 UDP 流量，实现精准分流，保证不同应用的最佳性能。
Nekoray 能否同时运行多个代理？
Nekoray 支持多配置运行。用户可在“多配置模式”下同时开启不同协议或节点的代理，程序会自动创建多个监听端口，方便多应用或分流测试使用。
Surfboard 的策略组可以根据延迟自动切换节点吗？
支持。Surfboard 可定期测速节点延迟，并结合策略组自动选择延迟最低的节点。这样用户无需手动切换节点即可保持高速访问，提高网络体验和稳定性。
WinXray 的内核能否替换？
可以。WinXray 的内核文件独立存在于程序目录中，用户可替换为最新的 v2ray-core 或 xray-core 文件以获取更好的性能与协议支持，但需保持版本兼容。
Sing-box 的域名规则如何实现分流？
Sing-box 支持按域名匹配流量，将不同网站或应用流量分配到指定节点。结合策略组自动切换低延迟节点，可优化访问速度，保证网络稳定性并提高整体体验。
Clash 的 Proxy Group 能实现哪些策略？
Proxy Group 可以定义多种策略，如按延迟选择、随机、轮询或手动选择节点。结合分流规则，可实现针对不同网站或应用的智能代理，提高网络体验。
V2Ray QUIC 传输适合什么网络环境？
QUIC 基于 UDP 支持多路复用和零握手，降低延迟，提高吞吐量。适合高丢包、不稳定网络环境，可保证视频、游戏和即时通讯流量的稳定性和流畅体验。
如何使用 Sing-box 配合 Clash Rules 实现精细分流？
将 Clash 的 rules 段提取出来转换为 Sing-box 的 routing 格式，或直接引用 rule-providers 源。Sing-box 的路由引擎支持更复杂的条件匹配，可结合 GeoIP 与 keyword 实现精细分流策略。
为什么使用 CDN 后仍然能被封？如何规避？
若 CDN 回源 IP 或配置不当可能泄露真实服务器信息，或者 CDN 本身被目标防火墙识别。应启用完整的反向代理配置、隐藏源站 IP 与禁用不必要的服务，必要时使用多级中转或更换 CDN 供应商。
Mihomo 节点分组如何自动切换？
用户可按延迟、地区或用途将节点分组，并启用策略组。系统会根据节点健康状况和延迟自动切换到最佳节点，保证网页浏览、视频播放和游戏连接稳定顺畅。
